# JudgeMe
-A website where a cat judges/roasts your desicions or questions. It is created to entertain people and it means no harm.

What does Judge me do?
-user inputs a decision, action or question = gemini API playes a role of cat and sends response

HTML, CSS, Javascript, Nodejs, Gemini API, Express is used to create it.

How to use it?
SIMPLE PROCESS:
go on this link = https://judgeme-1.onrender.com

LONG PROCESS :
clone it using this repo.
create an gemini api key and make .env file and add your API key in GEMINI_API_KEY variable.
